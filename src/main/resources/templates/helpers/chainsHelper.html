Les sous-chapitres


Un sous-chapitre est une suite ordonnée de groupes de règles.

Il permet de regrouper un ensemble de règles qui viennent logiquement ensemble, dans le but de contrôler un aspect donné d’un type d’objet.

L’ensemble des sous-chapitres permettant d’auditer un objet constitue le chapitre objet. Le sous-chapitre est un artifice qui évite de dupliquer les séquences de règles identiques.

Un sous-chapitre est actif quand tous les groupes de règles qui le composent sont actifs.
