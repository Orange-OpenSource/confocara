Les chapitres objets


Un chapitre objet permet d’auditer un objet.

C’est l’élément unitaire qui compose le référentiel. Il est constitué d’une liste ordonnée de sous-chapitres composés de groupes de règles.

Une fois finalisé, il est publié par son auteur et une version est automatiquement générée par l’application. Toute modification portera ensuite sur la version de travail.

Il faut que tous les sous-chapitres soient actifs pour pouvoir publier un chapitre.
