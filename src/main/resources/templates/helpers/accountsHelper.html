Les comptes utilisateurs


Liste des comptes utilisateurs

Cliquer sur le menu Accounts du bandeau gauche. Vous accéder à la liste des utilisateurs

Pour consulter le détail d’un compte, cliquer sur l’identifiant. Les pictos à droite permettent de modifier un compte ou de le supprimer.



Créer un Utilisateur

Cliquer sur Create account.

« * » indique les champs obligatoires. Par défaut, le rôle est contributeur. Il permet d’intervenir dans la constitution des jeux de règles. Le rôle administrateur permet de configurer le service et de gérer les utilisateurs.
