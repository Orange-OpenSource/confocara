Les types de profil


C’est une liste de valeurs possibles, avec un picto associé à chaque valeur. Le bouton « Ajouter un type de profil » permet de créer de nouvelles valeurs. Ces valeurs peuvent être supprimées tant qu’il n’y a pas de règle associée.

L’impact d’une règle non respectée est analysé par type de profils.

Par exemple :

  • impact = bloquant pour profil = fauteuil roulant
  • impact = grave pour type de profil = risque incendie

Un référentiel travaille avec une liste de valeurs de types de profils déterminée et associée à la catégorie de règles.
