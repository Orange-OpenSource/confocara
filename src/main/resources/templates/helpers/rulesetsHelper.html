Les référentiels


C’est l’ensemble des objets à auditer, sélectionnés via leurs chapitre objet publiés.

Le nom du référentiel doit être parlant pour l’auditeur. L’auteur du référentiel sélectionne les chapitres objets disponibles et qu’il souhaite mettre dans son groupe. Seules les versions des chapitres les plus récentes sont présentées.

Il est recommandé que l’auteur contrôle bien son référentiel avant de le publier.

La génération du référentiel publié peut être longue, il faut patienter !

La version du référentiel est générée automatiquement. Le jeu de données publié sera alors visible et téléchargeable depuis l’application OCARA.

Les chapitres des sous-objets associés qui ne sont pas directement intégrés sont eux versionnés automatiquement et placés dans le référentiel.
